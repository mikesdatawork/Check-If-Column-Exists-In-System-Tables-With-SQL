## About-Process



![If Column Exists]( https://mikesdatawork.files.wordpress.com/2017/10/image001.png "Check If Column Exists With SQL")
 
<p>If you have need to check if a column exists under System Tables (In this example we'll look for the [containment] column which shows support for contained databases) here's a quick bit of SQL logic that might help.</p>   


## SQL-Logic
```SQL
use master;
set nocount on
 
-- check for contained databases column under sys.databases
if col_length('sys.databases', 'containment') is null
    begin
        select 'Containment has not been found.'
    end
    else
        select 'Containment has indeed been found.'
```
